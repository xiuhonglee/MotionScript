# AE Scripts

A collection of After Effects scripts for motion graphics and animation.

## Scripts

| Script | Description |
|--------|-------------|
| `HandwritingNumbers.jsx` | 将数字转换为手写动画效果的面板插件。支持多位数字分组预合成，可调节间距和时序。 |
| `LinearCombinationControl.jsx` | 创建向量线性组合控制系统，三个控制点沿向量移动，始终满足 k₁a₁ + k₂a₂ + k₃a₃ = b。 |
| `VectorArrow.jsx` | 创建可控制的向量箭头，支持起点/终点、箭头大小、颜色和绘制进度动画。 |
| `LinearEquationsDemo.jsx` | 创建两条过原点的直线，通过 Slider 控制斜率（保留一位小数），附带控制点标记。 |
| `lineWithControlPoint.jsx` | 创建直线 2x-3y=0 及可拖动控制点。控制点沿直线移动，垂直虚线投影到 x 轴。 |

## Usage

1. Open After Effects
2. Run the script via `File > Scripts > Run Script File...`
3. For `HandwritingNumbers.jsx`, a panel UI will appear for configuration
4. For other scripts, select a composition first before running
